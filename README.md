# ML BOM Ancillary Prediction

This project builds a Machine Learning model to predict ancillary SKUs for telecom primary SKUs using historical BOM data.

## Problem
In telecom network deployments, selecting correct ancillary components for a primary SKU is often manual and error-prone.

## Solution
A machine learning model predicts recommended ancillary SKUs based on historical BOM datasets.

## Dataset
The dataset used for training is not included in this repository due to confidentiality and company data protection policies.

## Features
• Data preprocessing and feature engineering  
• Random Forest based prediction model  
• Automated BOM recommendation pipeline  

## Tech Stack
Python  
Pandas  
NumPy  
Scikit-learn  

## Run the Project

Install dependencies

pip install pandas numpy scikit-learn

Run model

python ancillary_recommendation_pipeline.py
